## üß© Description

This repository contains Java solutions for the **Advent of Code** programming puzzles.

The goals of this repo are:

- To practice **problem-solving, algorithms, and data structures** using real puzzle-style problems.
- To maintain a clean, navigable archive of solutions that you (and others) can learn from later.
- To experiment with different implementation styles, optimizations, and refactors over time.

**Who is this for?**

- Students and self-learners looking to see **Java approaches** to AoC puzzles.
- Developers wanting to **compare logic** and **solution structure** with their own.
- Anyone who enjoys puzzles and wants a reference implementation.
